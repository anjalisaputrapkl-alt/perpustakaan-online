Management System Perpustakaan Online (Multi-tenant)
Deskripsi Proyek
Banyak institusi pendidikan yang memiliki fasilitas perpustakaan fisik namun belum didukung oleh sistem manajemen digital yang memadai. Proyek ini dikembangkan dengan misi untuk mempermudah sekolah dalam mendigitalisasi dan mengelola sistem perpustakaan mereka secara mandiri.

Aplikasi ini menggunakan arsitektur Multi-tenant, yang memungkinkan penggunaan satu platform oleh berbagai sekolah namun tetap menjamin keamanan dan isolasi data untuk masing-masing instansi.

Fitur Unggulan
Sistem Multi-tenant: Pemisahan akses dan data antar sekolah yang berbeda.

Manajemen Inventaris: Pengelolaan database buku, kategori, dan stok secara real-time.

Sirkulasi Terpadu: Pencatatan otomatis untuk transaksi peminjaman dan pengembalian buku.

Manajemen Keanggotaan: Pengelolaan data siswa dan staf sekolah dalam satu dasbor.

Panduan Instalasi
Ikuti langkah-langkah berikut untuk menjalankan aplikasi di lingkungan lokal (local development):

Clone Repository
Lakukan kloning repositori ke direktori lokal Anda:

Bash
git clone https://github.com/username/repository-name.git
Konfigurasi Folder
Ubah nama folder proyek hasil kloning menjadi: perpustakaan-online

Konfigurasi Database

Pastikan layanan MySQL Anda telah aktif.

Buat database baru dengan nama: perpustakaan_online

Import file database (SQL) yang tersedia ke dalam database yang baru saja dibuat.
