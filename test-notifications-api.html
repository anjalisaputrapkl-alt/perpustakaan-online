1. GET - Daftar Notifikasi

Cara Kerja: Mengambil semua notifikasi berdasarkan sort filter (latest/oldest/unread)
fetch('api/notifications.php?action=list&sort=latest') .then(r => r.json()) .then(d => console.log(d.data))

2. GET - Detail Notifikasi

Cara Kerja: Mengambil detail 1 notifikasi + otomatis mark sebagai dibaca
fetch('api/notifications.php?action=detail&id=1') .then(r => r.json()) .then(d => console.log(d.data))

3. POST - Tandai Sebagai Dibaca

Cara Kerja: Mengubah status notifikasi menjadi "sudah dibaca" (1)
const fd = new FormData(); fd.append('id', 1); fetch('api/notifications.php?action=mark_read', { method: 'POST', body: fd }).then(r => r.json())

4. POST - Tandai SEMUA Sebagai Dibaca

Cara Kerja: Mengubah status semua notifikasi belum dibaca menjadi dibaca
fetch('api/notifications.php?action=mark_all_read', { method: 'POST' }).then(r => r.json())

5. POST - Hapus Notifikasi

Cara Kerja: Menghapus notifikasi dari database
const fd = new FormData(); fd.append('id', 1); fetch('api/notifications.php?action=delete', { method: 'POST', body: fd }).then(r => r.json())

6. GET - Hitung Notifikasi Belum Dibaca

Kegunaan: Untuk menampilkan badge di navbar
fetch('api/notifications.php?action=unread_count') .then(r => r.json()) .then(d => { document.getElementById('badge').innerText = d.unread_count; })

7. GET - Statistik Notifikasi

Cara Kerja: Menampilkan statistik per jenis notifikasi (telat, peringatan, dll)
fetch('api/notifications.php?action=stats') .then(r => r.json()) .then(d => console.log(d.data))
